### Propriedade para estilos customizados

<Space size={3} />

Utilize a propriedade `custom` para escrever qualquer regra CSS.

<Space size={6} />

### Exemplo 1

`bgColor: primary2`

`*color: rgba(255, 200, 255, 1);`

`*padding: 10px`

<pre>
  <CodeBlock>{`<Block bgColor='primary2' custom='
    color: rgba(255, 200, 255, 1);
    padding: 10px;
  '></Block>`}</CodeBlock>
</pre>

<Space size={3} />

<ThemeProvider theme={theme}>
  <Block 
    bgColor='primary2'
    custom={`
      color: rgba(255, 200, 255, 1);
      padding: 10px;
    `}>
    <span>item1</span>
    <span>item2</span>
    <span>item3</span>
  </Block>
</ThemeProvider>
