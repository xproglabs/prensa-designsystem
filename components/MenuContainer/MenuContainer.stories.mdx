# Menu Lateral (SideMenu)

<Space size={4} />

<CodeBlock>{`import {MenuContainer} from 'prensa';`}</CodeBlock>

<Space size={3} />

SideMenu é um esqueleto para a construção de menus laterais no front. Ele carrega algumas regras de comportamento e estilos e funciona a partir de um gatilho (mudança de estado).

É composto por um block e dois containêres, o primeiro é responsável por ser o conteúdo. O segundo é a área de clique externa (backdropClick) e permite o fechamento do menu.

### Como usar?
Para utilizar o componente, precisamos passar valor para a props `open` e uma função para `close`.
No exemplo abaixo as funções `handleClick` e `handleClose` realizam as manipulações de estado para abrir | fechar o menu lateral.

<pre>
  <CodeBlock>{`
    import React, {useState} from 'react';
    import {MenuContainer} from 'prensa';

    const Component = () => {

      /*  Estado inicial false  */
      const [menu, setMenu] = useState(false);

      /*  Função que aciona menu (state = true)  */
      const handleClick = () => {
        setMenu(true);
      };

      /*  Função que fecha o menu (state = false)  */
      const handleClose = () => {
        setMenu(false);
      };
      
      return (
        <div>
          /*  Botão gatilho para acionar o menu lateral  */
          <button onClick={() => handleClick()}>Abrir Menu</button>

          /* Componente Menu Lateral */
          <MenuContainer open={menu} close={handleClose}>
            {...content}
          </MenuContainer>
        </div>
      )
    }
  `}</CodeBlock>
</pre>

<Space size={6} />

### Propriedades do componente (props)
- O componente pode ser posicionado à esquerda **`left`** ou direita da tela **`right`**, usando a props **`menuAnchor`**.
- É possível alterar o seu tamanho máximo na tela usando a props **`menuSize`**, usando uma string '250px'.
- É possível alterar os paddings do conteúdo através das props **`px`** e **`py`**.
- Usando a props **`bgColor`** é possível alterar a cor do fundo do conteúdo.
