# Block / Margins

<Space size={3} />

Block é um componente baseado em uma `<div>` para entregar posicionamentos blocados na tela.

<Space size={6} />

### Propriedade para definir margins

<Space size={3} />

Utilize a propriedade `mt` para definir valor geral para o margin no topo

Utilize a propriedade `mb` para definir valor geral para o margin embaixo

Utilize a propriedade `ml` para definir valor geral para o margin na esquerda

Utilize a propriedade `mr` para definir valor geral para o margin na direita

Utilize a propriedade `mx` para definir valor geral para o margin na direita e na esquerda

Utilize a propriedade `my` para definir valor geral para o margin no topo e embaixo

É possível também usar os valores com os múltiplos de 8, use por exemplo `my={5}` para aplicar o valor de `40px`, por exemplo.

<Space size={6} />
