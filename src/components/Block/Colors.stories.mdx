# Block / Colors

<Space size={3} />

Block é um componente baseado em uma `<div>` para entregar posicionamentos blocados na tela.

<Space size={6} />

### Propriedades para cores

Pode ser utilizado propriedades de cores para background `bgColor` e cor primária da fonte `fontColor`.

Os valores aceitáveis são os contidos no styleguide / theme do design system.

<Space size={6} />

### Exemplo 1:

`bgColor='primary1'`

`fontColor='white'`

<pre>
  <CodeBlock>{`<Block bgColor='primary1' fontColor='white'>children</Block>`}</CodeBlock>
</pre>

<Space size={3} />

<ThemeProvider theme={theme}>
  <Block bgColor='primary1' fontColor='white' custom={`padding: 10px;`}>
    Texto
  </Block>
</ThemeProvider>
