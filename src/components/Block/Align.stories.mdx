# Block / Align

<Space size={3} />

Block é um componente baseado em uma `<div>` para entregar posicionamentos blocados na tela.

<Space size={6} />

### Propriedades de alinhamento

<Space size={3} />

Utilize a propriedade `align` para alternar entre o flex `row` e `column`

Utilize a propriedade `alignx` para definir a posição no eixo x podendo alternar entre `left`, `center`, `right`, `evenly` e `between`.

Utilize a propriedade `aligny` para definir a posição no eixo y podento alternar entre `top`, `middle`, `bottom`, `evenly` e `between`.

<Space size={6} />

### Exemplo 1: 

`row left top`

<pre>
  <CodeBlock>{`<Block align='row' alignx='left' aligny='top'></Block>`}</CodeBlock>
</pre>
