# Block / Fonts

<Space size={3} />

Block é um componente baseado em uma `<div>` para entregar posicionamentos blocados na tela.

<Space size={6} />

### Propriedade para font-family

<Space size={3} />

Utilize a propriedade `typography` para definir o font-family do componente.

<Space size={6} />

### Exemplo 1

`typography: primary`

<pre>
  <CodeBlock>{`<Block typography='primary'></Block>`}</CodeBlock>
</pre>

<Space size={3} />

<ThemeProvider theme={theme}>
  <Block 
    bgColor='primary2'
    fontColor='white'
    typography='primary'
    custom={`
      padding: 10px;
      span {
        padding: 10px;
      }
    `}>
    Texto 01 com typography primária aplicada.
  </Block>
</ThemeProvider>
