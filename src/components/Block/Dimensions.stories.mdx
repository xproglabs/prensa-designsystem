# Block / Dimensions

<Space size={3} />

Block é um componente baseado em uma `<div>` para entregar posicionamentos blocados na tela.

<Space size={6} />

### Propriedade para definir dimensões

<Space size={3} />

Utilize a propriedade `height` ou `width`

<Space size={6} />

### Exemplo 1

`height: '20px'`

`width: '100px'`

<pre>
  <CodeBlock>{`<Block width='100px'></Block>`}</CodeBlock>
</pre>

<Space size={3} />

<ThemeProvider theme={theme}>
  <Block 
    bgColor='primary2'
    fontColor='white'
    width='100px'
    height='20px'>
    Texto simples
  </Block>
</ThemeProvider>
